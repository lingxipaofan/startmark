# Startmark

将 Chrome 浏览器的书签栏变成新标签页启动面板，打开新标签即见所有收藏，一键直达。

## 功能

- **接管新标签页**：打开新标签时直接展示书签导航，替代 Chrome 默认空白页。
- **瀑布流布局**：书签按文件夹分组，以多列瀑布流排布，不同高度的分组自然对齐，减少空白。
- **一键打开**：点击书签直接在当前标签页跳转，无多余的选中步骤。
- **自定义拖拽排序**：拖拽书签和文件夹即可自由调整顺序，支持跨文件夹移动，排序结果持久化到本地。提供「自定义」排序模式，与 Chrome 原顺序、字母排序并列。
- **内联编辑弹窗**：重命名书签、编辑 URL、新建文件夹均使用专属模态弹窗，替代浏览器原生 `prompt()` 弹窗。
- **多维度排序**：支持 Chrome 原顺序、自定义拖拽顺序、名称 A-Z / Z-A 四种排序，切换后自动记住偏好。
- **界面缩放**：支持整体界面缩放（75% ~ 125%），偏好自动保存。
- **精简标题**：可开启标题简化，自动隐藏书签中的通知数字、重复站名和冗余后缀，不影响原始数据。
- **设置面板**：主题（明/暗）、界面语言、标题精简开关统一收入右上角齿轮按钮激活的弹窗中。
- **右键菜单**：书签支持在新标签页 / 新窗口打开、重命名、编辑 URL、删除；页面空白处右键可新建文件夹。
- **删除可撤销**：删除书签后可恢复，保留原有目录位置。
- **失效链接检测**：可主动检测当前页面书签的链接状态，失效提示低调不打扰。

## 开发

```bash
npm ci
npm run dev
npm run build
npm test
```

技术栈：WXT · React · TypeScript · Vite

## 安装

构建后在 Chrome 打开 `chrome://extensions/`，开启「开发者模式」，加载 `.output/chrome-mv3` 目录即可。

更新代码重新构建后，点击「重新加载」生效。
