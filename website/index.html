✨ 功能特性

✨ Features

🗂️

导航模式 / 列表模式

Grid & List View

多列卡片式导航，或传统的左侧树形文件夹 + 右侧列表，自由切换

Multi-column card grid or classic sidebar-tree + list view, switch freely

📁

拖拽整理

Drag & Drop

书签和文件夹均可拖拽移动，支持多选批量拖拽到目标文件夹

Drag bookmarks and folders to reorganize, supports batch drag with multi-select

🔍

全文搜索

Full-text Search

按标题和 URL 实时搜索，⌘F 快捷键快速聚焦

Real-time search by title and URL, ⌘F to focus

🌙

深色模式

Dark Mode

一键切换深色/浅色主题，自动记忆你的偏好

Toggle dark/light theme, remembers your preference

📂

一键打开全部

Open All

右键文件夹 → 打开全部，一次性打开文件夹内所有书签

Right-click a folder → Open All, open every bookmark in new tabs

↩️

撤销删除

Undo Delete

误删了书签或文件夹？Toast 通知栏一键撤销恢复

Accidentally deleted something? One-click undo from the toast notification

🕐

按时间排序

Sort by Time

按收藏时间分组查看，最早收藏的在前面，帮你发现尘封的书签

Group bookmarks by creation time, oldest first — rediscover forgotten gems

⌨️

键盘快捷键

Keyboard Shortcuts

⌘F 搜索 · ⌘A 全选 · Delete 删除 · Esc 关闭菜单

⌘F search · ⌘A select all · Delete remove · Esc close menus

📥 安装方法

📥 Installation

  1. 下载 ZIP 并解压到本地文件夹
  2. 打开 Chrome，进入 chrome://extensions
  3. 开启右上角的"开发者模式"
  4. 点击左上角"加载已解压的扩展程序"，选择解压后的文件夹
  5. 点击浏览器工具栏的 🔖 图标开始使用
  1. Download ZIP and extract to a local folder
  2. Open Chrome and go to chrome://extensions
  3. Enable "Developer mode" in the top-right corner
  4. Click "Load unpacked" and select the extracted folder
  5. Click the 🔖 icon in the toolbar to start using Pinmark
